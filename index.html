Working Directory

El area de trabajo corresponde a la carpeta donde inicializamos el repositorios de git. Rara vez vamos a etar utilizando la linea de comandos es basicamente donde vamos a crear editar y eliminar los diferentes archivos de nuestro proyecto

  • DIR || LS muestra el contenido de la carpeta actual
  • MKDIR < namecrea una carpeta con el nombre especificado
  • CD < PATH>cambia la carpeta actual por la seleccionada
  • GIT INITinicializa un repositorio de git en la carpeta actual
  • RM -RF .GITelimina el repositorio de manera forzada
